The project can be build and run via the Makefile and the following commands:

To build and run the initial part of Assignment 9 (memory manager with 256 frames):

make run_256

To build and run the modified part of Assignment 9 (memory manager with 128 frames):

make run_128

To build and run the part of Assignment 9 given in course homework (which builds on the 128-frame version above, plus adds a page written indicator):

make run_rw
